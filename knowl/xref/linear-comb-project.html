Project 2.1.2. Linear Combinations and Spanning.

As we have seen by the operations we have defined for vectors, we can create new vectors using combinations of scalar multiplication and addition of existing vectors. In this lab you will explore both the algebraic and geometric combination of such vectors and develop an intuition for how algebraic operations relate to geometric representations of the same ideas.

If we start with a certain set of fixed vectors and look at all of the vectors we can get (or reach) with just linear combinations of these vectors (\(a\mathbf{v}+b\mathbf{w}\)), it is like asking what objects can we grab with our arms? Thus we call the set of all vectors that we can reach with linear combinations of a certain set of vectors the Span of those vectors (analogous to our “arm span”). In this case we would denote the set of all vectors that we can obtain from linear combinations of \(\mathbf{v}\) and \(\mathbf{w}\) by \(Span\left\{\mathbf{v},\mathbf{w}\right\}\text{.}\)
