Theorem 2.1.13.

Given a linearly independent set of vectors, \(\left\{\mathbf{v}_1, \ldots, \mathbf{v}_k\right\}\text{,}\) and any vector, \(\mathbf{w} \in Span\left\{\mathbf{v}_1, \ldots, \mathbf{v}_k\right\}\text{,}\) then the scalars, \(c_1, c_2, \ldots, c_k\text{,}\) needed to reach \(\mathbf{w}\) are unique.

Proof.

Suppose we have a linearly independent set of vectors, \(\left\{\mathbf{v}_1, \ldots, \mathbf{v}_k\right\}\text{,}\) and any vector, \(\mathbf{w} \in Span\left\{\mathbf{v}_1, \ldots, \mathbf{v}_k\right\}\text{.}\) Further, suppose we also have \(c_1, c_2, \ldots, c_k\text{,}\) such that \(c_1\mathbf{v}_1+c_2\mathbf{v}_2+ \cdots +c_k\mathbf{v}_k=\mathbf{w}\text{.}\) If \(c_1, c_2, \ldots, c_k\) are not unique, then there exists another set of scalars, \(a_1, a_2, \ldots, a_k\text{,}\) such that \(a_1\mathbf{v}_1+a_2\mathbf{v}_2+ \cdots +a_k\mathbf{v}_k=\mathbf{w}\text{.}\) Subtracting these equations, we get \(\left(c_1-a_1\right)\mathbf{v}_1+\left(c_2-a_2\right)\mathbf{v}_2+ \cdots +\left(c_k-a_k\right)\mathbf{v}_k=\mathbf{0}\text{.}\) Since \(\left\{\mathbf{v}_1, \ldots, \mathbf{v}_k\right\}\) are linearly independent, this means that \(c_1-a_1=0\text{,}\) \(c_2-a_2=0, \ldots, c_k-a_k=0 \Rightarrow c_1=a_1, c_2=a_2, c_k=a_k\text{.}\) In otherwords, the scalars are all equal and thus unique.
